Whether you’re doing a Bungee Drop for AP Physics class or the Science Olympiad event, you’ll need to know a decent amount of physics. This entire guide is intended for those who have no experience with physics, but the first few sections can be skipped by those who know its basics. But before we get into the math and physics, let’s review the event parameters for Bungee Drop. The Rulebook does not specify how you design your elastic band, as long as it does not consist entirely of an inelastic material (which would ruin the point of the event). Because we want to minimize the number of variables we deal with, it’s best to have a fixed elastic portion and an inelastic portion that we adjust based on the drop distance and mass. The longer the drop distance, and the greater the mass, the further our elastic material is going to stretch, so if we keep the elastic material the same, we will have to change the string of fixed length.
